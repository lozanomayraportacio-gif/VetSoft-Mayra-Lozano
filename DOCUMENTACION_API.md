\# 📘 Documentación API - VetSoft



\## 🐾 Proyecto Formativo VetSoft



Sistema web veterinario desarrollado con Java, Servlets, MySQL y React para la gestión clínica y administrativa de mascotas.



El sistema fue diseñado para facilitar la gestión veterinaria mediante módulos web integrados.





\# 🔐 API Login



\## Endpoint



/LoginServlet



\## Método



POST



\## Descripción



Permite validar el acceso de usuarios al sistema.



\## Parámetros



| Parámetro | Tipo | Descripción |

|-----------|------|------------|

| usuario | String | Usuario del sistema |

| contraseña | String | Contraseña del usuario |
